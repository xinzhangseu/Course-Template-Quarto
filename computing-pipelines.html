Model fitting

Fit model:

penguins_fit <- linear_reg() %>%
  set_engine("lm") %>%
  fit(body_mass_g ~ flipper_length_mm, data = penguins)

Tidy model output:

tidy(penguins_fit)
# A tibble: 2 × 5
  term              estimate std.error statistic   p.value
  <chr>                <dbl>     <dbl>     <dbl>     <dbl>
1 (Intercept)        -5781.     306.       -18.9 5.59e- 55
2 flipper_length_mm     49.7      1.52      32.7 4.37e-107

Format model output as table:

tidy(penguins_fit) %>%
  kable(digits = 3)
term estimate std.error statistic p.value
(Intercept) -5780.831 305.815 -18.903 0
flipper_length_mm 49.686 1.518 32.722 0
